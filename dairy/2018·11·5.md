## element ui

[地址](http://element.eleme.io/#/zh-CN)

- vue-cli 初始化

> npm yarn -g
>
> vue init webpack project_name
>
> npm i element-ui -S

快速上手

```js
import Vue from 'vue';
import ElementUI from 'element-ui';
import 'element-ui/lib/theme-chalk/index.css';
import App from './App.vue';

Vue.use(ElementUI);

new Vue({
  el: '#app',
  render: h => h(App)
});
```
