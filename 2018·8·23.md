## 假的基础教程

首先你需要安装 nodejs and npm
如果你安装的是 4.0以上版本的话，你就不用单独安装npm了，一般会已经附带了该功能
现在你安装了 npm 就可以 开始安装 vue和vue-cli了
>npm install -g vue
>
>npm install -g vue-cli
>
>vue init webpack you_project
>
>cd you_project
>
>npm install
>
>npm run dev
